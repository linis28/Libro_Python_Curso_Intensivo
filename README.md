# 📘 Libro / Curso Intensivo de Python

## 📌 Descripción

Este repositorio contiene el desarrollo de un libro/curso intensivo de Python, con explicaciones teóricas, ejemplos prácticos y ejercicios tipo universitario.

El objetivo es fortalecer habilidades en programación con Python desde un enfoque progresivo, práctico y académico.

---

## 🎯 Objetivos

- Aprender Python desde cero hasta nivel intermedio
- Comprender estructuras fundamentales del lenguaje
- Resolver ejercicios prácticos tipo examen universitario
- Aplicar conceptos en problemas reales

---

## 📚 Contenido del curso

- Introducción a Python
- Variables y tipos de datos
- Operadores
- Estructuras de control (if, for, while)
- Funciones
- Listas, tuplas y diccionarios
- Programación orientada a objetos
- Ejercicios prácticos
- Material tipo libro académico

---

## 🧰 Tecnologías usadas

- Python 3.x
- Jupyter Notebook (si aplica)
- Git y GitHub
- (Opcional) LaTeX para material teórico
